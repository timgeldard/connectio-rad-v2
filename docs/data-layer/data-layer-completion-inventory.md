## How to read this table

- **Adapter mode** = what mode the frontend adapter uses in production deployment (not mock dev mode).
- **Source verification state** = what has been confirmed via Databricks CLI or direct DDL inspection. "not-verified" means no evidence exists; it does not mean the source is wrong.
- **Not found in audit** = searched for but not located; may exist under a different name.
- **Generated Python model** = exists in `apps/api/contracts/generated.py` (auto-generated from Zod schema via contracts.json).
